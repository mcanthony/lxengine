Tools & Technologies

JQuery Plug-ins

Seamless Image View v0.2.0
Replaces a standard <IMG> element with a tiled view of the image that can be manipulated via zooming or panning.
Image Effects v0.0.0
Runs various configurable image filters on <IMG> elements within a page. Grayscale, desaturation, and hue shifts are example operations.
Fire Effect v0.1.0
A classic fire effect implemented with an HTML5 <CANVAS> element.
Color Wheel v0.0.0
Draws a color wheel with various options.

Research & Labs Projects

Simulation Engines

LxEngine WIP
A desktop visualization & simulation engine based on a XML and Javascript technologies, while written in C++ for performance.
LxWeb WIP
The entirely web-based version of LxEngine. Written entirely in Javascript.

Component Technologies

LxLang WIP
A simple, C-like language for developing low-level, "building-block" libraries that can be retargeted to native Javascript, C++, and GLSL to allow for code sharing between multiple implementations.
Shader Builder WIP
A web-based shader builder targeting GLSL, C++ based ray-tracers, and Javascript ray-tracers with presets & customization for immediate compatiblity with various engines.
BlenderLoad WIP
A web-based Javascript Blender file (.blend) importer and translator. Targets WebGL and custom targets.

Asset and Information Management

PIM0 WIP
A personal information management system. In research.
AIMS WIP
Asset Index Management System (AIMS). A web-based, "type-aware" overlay of a git respository with automatic asset previews, data format conversions, and packaging.

Proof-of-Concept Applications

Ray Tracer WIP
A web-based ray-tracer demonstrating the graphical capabilities of LxWeb and various graphical component technologies.
Gravity Tide WIP
A simple proof-of-concept game demonstrating LxWeb. Focuses on animations, audio, and timelining.
Night's Return WIP
A proof-of-concept game demonstrating LxEngine on the desktop with a particular focus on multi-user and networking support.

The research and labs projects within Athile Technologies represent more ambituous, more volatile, and often less complete projects within the company.

Information available on these projects is highly subject to change.
