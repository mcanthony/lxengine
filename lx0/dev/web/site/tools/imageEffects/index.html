Image Effects

About

The imageEffect JQuery plug-in currently serves as a proof-of-concept and base for developing pixel-based image effects on <IMG> elements within a page. It is still in early development and supports only a minimal number of effects. Please provide feedback if you would like to see further development of this plug-in.
